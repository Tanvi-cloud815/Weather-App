Weather App

A modern weather application built using HTML, CSS, JavaScript, and OpenWeather API.

Features

- Search weather by city name
- Real-time temperature display
- Weather condition information
- Humidity details
- Wind speed information
- Weather icons
- Error handling for invalid cities
- Responsive and user-friendly interface

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
- Fetch API
- Async/Await

How to Run

1. Download the project files.
2. Open "script.js".
3. Add your OpenWeather API key:
   const apiKey = "YOUR_API_KEY";
4. Save the file.
5. Open "index.html" in your browser.

Project Description

This application fetches real-time weather data using the OpenWeather API and displays temperature, humidity, wind speed, and weather conditions for any searched city.
